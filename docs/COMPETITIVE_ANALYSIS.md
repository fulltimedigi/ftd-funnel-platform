# Competitive Analysis — interactive funnel builders

Teardown of the leading platforms, done to position this product. Full sourcing lives
in the session research; this is the actionable synthesis.

## The gap we take

Every competitor is either **(a) AI-builds-a-funnel-without-real-products**, or
**(b) real-products-but-only-in-Shopify-with-manual-logic**. **None** takes an arbitrary
brand URL → reads the real catalog → auto-authors a grounded diagnostic that recommends
real SKUs with justified reasons. That unclaimed position is our wedge — and it *is*
FullTimeDigi's founding discipline (real product, no fabrication), which the trust gate
already enforces in code.

## Table stakes (must match to be credible globally)

Multi-step one-question flow · conditional logic/branching · scoring (score-buckets
*and* weighted outcome-matching) · result pages · **mid-flow + partial lead capture** ·
native CRM/email + webhooks + Zapier · analytics (per-step drop-off, answer/result
distribution) · branding/white-label + custom domain + embed + hosted page ·
mobile-first + fast load · A/B testing · templates.

## Where leaders differentiate

Recommendation intelligence (naive buckets vs catalog-aware matching) · AI autobuild
depth · built-in post-conversion nurture · paid-traffic optimization · formula power.

## Our priority protocol (ordered by impact)

**Tier 1 — the wedge (Stage 1–2):** ① brand/URL → real catalog ingestion · ②
auto-derived decision axes → fact-based questions · ③ automatic answer→**real-SKU**
scoring · ④ explainable, answer-grounded recommendation · ⑤ lead capture with the
answer profile attached.

**Tier 2 — table stakes (Stage 3):** editable logic/scoring (human overrides AI) ·
CRM/email/webhooks pushing answers+score+recommended product · hosted page + embed +
custom domain · brand auto-styling from the ingested site · analytics dashboard.

**Tier 3 — parity/margin:** A/B testing · post-conversion nurture · calculation/quote
diagnostics · ranked multi-outcome · chat-refine editor · vertical templates as
cold-start fallback.

**Strategic one-liner:** *paste any brand URL → an AI reads the real catalog, derives
the decision, authors a grounded diagnostic that recommends real SKUs with reasons, and
captures the lead* — automating the one step (answer→real-product scoring) every current
tool leaves to humans.
